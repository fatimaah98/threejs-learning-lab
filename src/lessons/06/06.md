## ğŸ”¤ TextGeometry Basics

Three.js does not render text directly. Instead, text is converted into geometry using a font file (`.json`).

```js
const textGeometry = new TextGeometry('Hello World', {
  font,
  size: 0.2,
  depth: 0.1
})
```

### Important Properties:

* **font**: Loaded font data (required)
* **size**: Height of the text
* **depth**: Thickness of the text (extrusion)

The text uses `MeshStandardMaterial`, allowing it to react realistically to scene lighting:

```js

  color: 0xffffff,

```

This gives the text a slightly metallic and polished appearance.
