# Lesson 01: Scene & Camera Basics

In this lesson, I learned how to set up the **fundamental structure** of a Three.js scene.

### Key Concepts:

1. **Scene**
   - The main container for all objects, lights, and cameras.
   - Set the background color using `scene.background`.

2. **Camera**
   - Used to view the scene.
   - `PerspectiveCamera(fov, aspect, near, far)` creates a realistic perspective.
   - **Parameters explained:**
     - `fov` (Field of View): the vertical viewing angle in degrees. Higher value â†’ wider view. Example: `75`.
     - `aspect`: the ratio of width to height of the display (`window.innerWidth / window.innerHeight`).
     - `near`: the closest distance the camera will render. Objects nearer than this are not visible. Example: `0.1`.
     - `far`: the farthest distance the camera will render. Objects beyond this are not visible. Example: `1000`.
   - Positioned at `camera.position.z = 5` to make objects visible.

3. **Renderer**
   - `WebGLRenderer` handles rendering the scene to the DOM.
   - `renderer.setSize(window.innerWidth, window.innerHeight)` sets the viewport.
   - Append `renderer.domElement` to `document.body` to display it.

4. **Rendering**
   - `renderer.render(scene, camera)` draws the scene once.

### Outcome:
- Successfully displayed an empty black scene with a visible camera perspective.
- Prepared the environment for adding geometries, materials, and lights in the next lessons.
