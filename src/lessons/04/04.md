## ðŸ”„ Object Rotation

Objects in Three.js can be rotated on three axes: `x`, `y`, and `z`.

```js
cube.rotation.x += 0.05
```
- This rotates the cube continuously on the X axis.

- Rotation values are measured in radians.

- Placing this inside the animation loop creates smooth motion.

## ðŸ§­ GridHelper

a visual helper that displays a grid on the scene
``GridHelper(size, divisions)``

```js
const grid = new THREE.GridHelper(10, 10)
```
### Why it's useful?
1. **Helps understand object positions and distances**
2. **Makes spatial orientation easier while learning**
3. **Acts like a 3D reference plane**

## ðŸ’¡ Lights
- AmbientLight: **``Provides uniform light without direction and shadow to prevent scenes from being completely dark``**
    - Properties: **AmbientLight(color, intensity)**
- PointLight: **``Emits light from a single point, Similar to a light bulb``**
    - Properties: **PointLight(color, intensity, distance)**
- DirectionalLight: **``Emits light in a specific direction, Similar to sunlight``**
    - Properties: **DirectionalLight(color, intensity)**

## ðŸŽ¨ Materials
Materials define how objects react to light.
### MeshBasicMaterial:
1. Does not reac to lights
2. Always fully visible
3. Useful for debugging or wireframes

### MeshLambertMaterial:
1. Reacts to light
2. Lightweight and simple

### MeshPhongMaterial:
1. Supports shiny highlights

### MeshStandardMaterial:
1. Physically based

### MeshPhysicalMaterial:
1. Advanced version of Standard material
2. Used for glass-like or transparent objects
